## 简述
这是博客内容管理系统，利用react + typescript + react-router + hooks + ant design编写而成。具体模块有

1. 登录、退出
2. 文章管理
   - 文章修改，删除
   - 添加文章
   - 草稿箱
3. 标签管理
   - 新增、修改、删除标签
4. 留言墙管理
   - 新增、修改、删除留言
5. 文章评论管理
   - 新增、修改、删除留言
6. 项目管理
   - 新增、修改、删除留言
7. 音乐管理
   - 新增、修改、删除音乐
   - 上传音乐文件、海报到七牛云

### Clone

````
git clone git@github.com:naihe138/naice-blog-admin.git
````

### Install

````
yarn --ignore-engines
````

### Dev

````
npm start
````

### Build

````
npm run build
````
